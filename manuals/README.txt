Taskpplication Directory Contents:
	Directories
	- src (source code)
	- lib (h2 jar)
	- build (executables)
	- out (Scene builder files)
	- docs (all Javadoc)
	- manuals (user's manual, testing manual, readme)
	- scripts (run and javadoc scripts)


How to run scripts from the command line:

1) Open a terminal in the Taskpplication directory.
2) Run the following command:
	Windows:
		scripts\Run.bat <PATH_TO_JAVAFX\lib>
	macOS/Linux:
		bash scripts/Run.sh <PATH_TO_JAVAFX\lib>
